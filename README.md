# Dash Data Visualization Project

## Overview
This project is a Dash-based interactive data visualization tool built using Python.

## Features
- Interactive graphs
- Data exploration using Pandas
- Simple UI using Dash

## Tech Stack
- Python
- Dash
- Pandas
- Plotly

## Setup Instructions
1. Clone repo
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python app.py

## Output
Interactive dashboard running on localhost
